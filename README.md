# 签到日历系统 (Sign-in Calendar)

记录每日签到情况，以日历形式展示连续签到天数与断签状态。

## 在线访问

**地址**：https://deepgreenuniverse.github.io/signin-calendar/

## 技术栈

- React 18 + Vite
- Tailwind CSS
- Lucide React（图标）
- date-fns（日期处理）
- localStorage（数据存储）

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:5177

## 构建

```bash
npm run build
```

产物输出到 `dist/`

## 功能

- ✅ 今日签到（按钮状态管理）
- 📅 月历视图（月份切换）
- 🔥 连续签到天数统计
- 📊 累计签到天数统计
- ⚠️ 断签日标记
- 🎨 连续签到高亮样式
